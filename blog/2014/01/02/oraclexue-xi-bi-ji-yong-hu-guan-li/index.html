基础权限概念

  • 系统权限:值用户对数据库的相关权限,比如建表修改表
  • 对象权限:用户对其他用户的数据对象访问的权限,比如创建的表 触发器 视图
    • select
    • insert
    • update
    • delete
    • all
    • crate index
    • ….
  • 角色概念:一个角色包含几个权限,为了方便赋予权限
    • 自定义角色:自定义角色
    • 预定义角色:oracle预设角色
      • connect角色:连接权限
      • resourceq角色:在表空间创表
      • dba角色:超级管理员
